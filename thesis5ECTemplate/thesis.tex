%%%%% English title %%%%%
\etitle{Heart Rate Estimation Based on Blind Source Separation}

%%%%% Abstract %%%%%
\eabstract{ % これ単体で複数ページにまたがる場合はエラーが出るので注意，アブスト内で改段落は禁止
Audio source separation is a technique for separating original sources from an observed mixture signal.
For example, it can be used for a pre-process of an automatic speech recognition system.
Audio source separation is utilized for separating a target speech signal and the other speech or background noise.
In particular, audio source separation that does not utilize any prior information about mixing system is called blind source separation (BSS).
In this thesis, IVA, ILRMA, , which are typical methods of BSS, and $t$-ILRMA are applied to the measurement signal to estimate the heart rate.
The experimental results show it was confirmed that breathing may affect the estimation results. Therefore, a filter was designed and applied as a pre-process. As a result of applying each method to the filtered measurement signal, we confirmed that $t$-ILRMA has the highest separation.
}

%%%%% 概要 %%%%%
\abstract{ % これ単体で複数ページにまたがる場合はエラーが出るので注意，アブスト内で改段落は禁止
音源分離とは，複数の音源が混合している観測信号から混合前の音源を推定する技術である．
この技術は，例えば音声認識の前段処理に用いられる．
目的話者の音声信号とその他の音声や背景雑音等の分離に音源分離が活用される．
特に，音源の混合系に関する事前情報を全く用いずに音源を分離する技術はブラインド音源分離（blind source separation: BSS）と呼ばれる．
本論文では，BSSの代表的手法であるIVAやILRMA, $t$-ILRMAを測定信号に適用し心拍推定を行う．
実験結果より，呼吸が推定結果に影響を与えている恐れがあることを確認した．そのため，前処理としてフィルタを設計して適用した．フィルタリングを行った測定信号に各手法を適用した結果$t$-ILRMAが最も分離制度が高いことを確認した．
}
